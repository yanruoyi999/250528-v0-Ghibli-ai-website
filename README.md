# Ghibli AI 图片生成器

基于 Next.js 开发的吉卜力风格 AI 图片生成器。

## 功能特性

- 🎨 吉卜力风格图片生成
- 📱 响应式设计
- 🔧 多种纵横比支持
- 💾 本地历史记录保存
- 📥 图片下载功能

## 技术栈

- Next.js 15
- TypeScript
- Tailwind CSS
- Radix UI
- Lucide Icons
